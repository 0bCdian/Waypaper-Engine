║  │  Complexity: O(n × 1440) operations per day                           │ ║

║  ┌────────────────────────────────────────────────────────────────────────┐ ║
║  │  function runTimeOfDayPlaylist() {                                     │ ║
║  │      // Initial setup (once)                                           │ ║
║  │      index = findClosestImageIndex()              // O(log n) binary   │ ║
